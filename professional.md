# Professional Documentation   
A collection of documentation created for professional environments. All documents are in PDF format, but were created using a combination of Microsoft Word, Canva, and Google Docs.  

## User Documentation
* [QTrak User Guide](./assets/docs/usermanual.pdf) - Complete redesign and modernization of an existing product user manual, including standardized formatting, improved organization, and accessibility focused updates.
* [Zebra Printer & Z-Routing Setup Guide](./assets/docs/zebra) - Configuration and troubleshooting guide covering Zebra printer setup, Browser Print, label calibration and Z-Routing workflows for administrators and personnel.
* [QTrak Asset Pods and Vending Guide](./assets/docs/assetpods.pdf) - Complete user guide for asset distribution and vending workflows including setup, and adding, removing and returning assets.  

## Configuration & Implementation  
* [QTrak API Contact Import Guide](./assets/docs/api) - Technical integration guide detailing API authentication, contact import workflows, required field mappings, and implementation examples for system administrators.
* [QTrak SAML SSO Configuration Guide](./assets/docs/sso.pdf) - Step-by-step configuration guide for SAML SSO with Microsoft Entra ID, including application setup, claims configuration, security groups and testing procedures.
